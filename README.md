# Enterprise Network Design Lab

## Overview
This project demonstrates a small enterprise network built using Cisco Packet Tracer.

The network includes VLAN segmentation, inter-VLAN routing, DHCP services, redundancy protocols, and security configurations.

## Features
- VLANs (Management, IT, Sales)
- Router-on-a-Stick (Inter-VLAN Routing)
- DHCP for automatic IP assignment
- HSRP for gateway redundancy
- OSPF for routing
- Extended ACL for access control
- DNS simulation

## Files Included
- networking_lab.pkt → Packet Tracer topology
- networking_lab.pdf → Full documentation

## What I Learned
- VLAN segmentation and trunking
- Routing between networks
- Redundancy using HSRP
- Dynamic routing with OSPF
- Network security using ACLs
